## Getting started
## Pointer
```kt
01  // down
10  // up
1   // left
0   // right
```

## Brush
```kt 
0001 // switch to the next brush | rectangle (default), circle
0010 // increase the brush size by 1 pixel
0011 // decrease the brush size
```

## Draw
```kt 
001 // white
010 // black
011 // yellow
100 // red
101 // blue
111 // green
```
> Note: also, you can add 0x before each command, it doesn't affect anything but looks cooler<br>
> example: 0x0001
